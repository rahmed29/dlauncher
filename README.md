# dlauncher

[prods.party](https://github.com/rahmed29/prods.party) repurposed into a dead
simple app launcher / directory for self-hosted web applications running on my
Dell Inspiron 3847 that I use as a home server.

![A screenshot of dlauncher in use](./screenshot.png)

Just run `deno -P server.js` and you'll be good to go!
